# Brighton & Hove Buses Telegram bot

A Telegram bot for the Brighton &amp; Hove Buses

Name of the bot: `bhbuses_bot`

To try it, type

```
briapaw
```

or

```
briapaw 7
```

For a full list of available codes, go to http://m.buses.co.uk/departure_stops.aspx?q=&find=find
